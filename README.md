# bgg_rec
Script to gather bgg recommendations for games

1. Input file format: JSON
[
  {
    "id": 174430,
    "name": "Gloomhaven",
    "year": 2017
  },
  {
    "id": 161936,
    "name": "Pandemic Legacy: Season 1",
    "year": 2015
  },
]
2. Output nodes file header: Id;Label;Year;Visited
3. Output edges file header: Source;Target
4. Use Gephi
